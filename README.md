# Gland Portfolio

Personal portfolio website of **Gland Jermano Blessed Siahaan**.

This repository contains my personal portfolio website, built as a learning project, academic web development project, and personal branding platform. The project represents my journey as an IT student who is currently building a foundation in web development, interface design, interaction, and basic web system development.

## Project Overview

**Gland Portfolio** is designed to introduce my identity, learning journey, skills, and web development projects. The website uses a clean visual style, interactive motion, scroll-based video experience, and bilingual language support.

The main purpose of this project is to create a professional digital identity while documenting my growth as a beginner web developer.
