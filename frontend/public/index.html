Bukan fase ketika saya paling banyak teori, tetapi fase ketika saya mulai paling banyak membangun.

Di semester 2, saya mulai lebih sering vibe coding. Bukan karena saya tidak paham flow atau alur, tetapi karena saya merasa lebih cepat belajar ketika ide langsung saya ubah menjadi sesuatu yang nyata.

Saya masih punya banyak waktu untuk explore. Karena itu, saya lebih senang mencoba berbagai ide, melihat hasilnya langsung di browser, lalu memperbaikinya sedikit demi sedikit sampai jadi website yang benar-benar terasa hidup.

Biasanya saya mulai dari membuat desain page terlebih dahulu. Setelah itu, saya gunakan AI untuk membantu mengubah desain tersebut menjadi kode, sambil tetap saya pahami struktur, logika, dan cara kerjanya.

Bagi saya, vibe coding bukan berarti asal jadi. Justru ini cara saya belajar sambil membangun, supaya setiap ide bisa benar-benar berubah menjadi pengalaman digital yang nyata.
